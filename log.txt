# 총 현금 = 모든 현금 계좌의 현재 잔액

# 순수입 = 저축 + 이자 - 지출
    # 투자는 제외. 현금화는 가능하지만 어쨌든 보유하고 있는 현금은 아니기 때문
    # 투자 탭은 수익률로 확인

# 계좌 타입 존재
    # 현금
        # 수입, 이자, 이동
        # 지출, 투자
    # 투자
        # 이동, 수익
        # 손실
    # 현금에서 투자로 이동한 금액은 순수입에 트래킹되지 않음
    # 월별 모은 금액은 얼마인데, 지금 현금 자산은 얼마이고, 총 자산 중에 투자가 차지하는 비율, 수익률
